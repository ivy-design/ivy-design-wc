# TimePicker 时间选择器

:::warning
依赖组件 `scrollbar`，请务必导入 `scrollbar` 组件
:::

## 常规用法

<ivy-time-picker></ivy-time-picker>

```html
<ivy-time-picker></ivy-time-picker>
```

## 禁用选项

通过 设置 `disabled` 属性来禁用时间选择器

<ivy-time-picker disabled></ivy-time-picker>

```html
<ivy-time-picker disabled></ivy-time-picker>
```
